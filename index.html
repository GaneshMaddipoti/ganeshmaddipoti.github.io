Key Features

  • Panning & Zooming: Navigate infinite canvas with precision.
  • Hierarchical Nodes: Nested containers for organizing complex structures.
  • Paths & Lines: Connect nodes flexibly to define relationships.
  • Save & Load: Persist your work effortlessly to/from JSON.
  • Keyboard Shortcuts: Boost productivity with Cut, Paste, Delete, Zoom shortcuts.
  • Rich Styling: Uniform design with Light/Dark themes supports.

Quick Start Guide

Toolbar & Creation

  • Select Tool: Default pointer for moving and editing.
  • Create Nodes: Click icon (Container/Box), then click on canvas to place.
  • Connect Nodes: Click Path/Line tool -> Click Start Node -> Click End Node.

Shortcuts & Actions

  • Move/Pan: Drag nodes to move. Drag empty space to pan.
  • Edit Text: Double-click text areas.
  • Save/Load: Ctrl+S / Ctrl+O (Cmd on Mac).
  • Clipboard: Ctrl+C, Ctrl+X, Ctrl+V for Copy/Cut/Paste.
  • Cancel/Deselect: Esc key.
