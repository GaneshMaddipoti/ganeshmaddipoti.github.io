Data Warehouse:
1) Integrated collection of data from different data sources
2) Subject oriented
3) Time variant - maintains historical data
4) Non-volatile - DW is stable between updates to it
5) Typically restructure and reorganize the data before putting into DW

Why we need to build DW
1) making data driven decisions
2) One source of truth

DW vs. Data Lake
1) DW is build on top of relational DB and multidimensional DB
2) Data lake is build in Big Data environment
