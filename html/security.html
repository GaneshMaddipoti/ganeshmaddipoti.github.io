Introduction
It’s a security protocol for Human to App and App to App authentication/authorization. (API authorization)
OLD Applications will integrate with other applications using - XML, SOAP, SAML, WS
New applications use JSON, HTTP APIs, OAuth and openID connect

Authentication : who they are
Authorization : what are permitted to
Existing protocol issues
Credentials Sharing : Amazon is taking IRCTC username and password to book the ticket.
Cookies : will be saved browser machine, and used by subsequent requests. (Gmail)
API Keys : Keys will be generated per API and given to apps. (Google Maps, weather API)
