Java

Introduction
Java was conceived by james gosling and his team at sun micro systems in 1991.
This language was initially called “OAK” , but was renamed to “JAVA” in 1995.
Java derives much of the syntax from ‘C’ and ‘C++’.
Translating a java program into byte code helps makes it much easier to run a program in a wide variety of environments,
the reason is, only the JVM need to implemented for each platform.
The details of jvm differ from platform-to-platform.

The java buzzwords
  • Simple
  • Object Oriented
  • Robust by exception handling and garbage collection
  • Multi Threaded
  • Portable, Architecture neutral, Interpreted
  • Distributed
